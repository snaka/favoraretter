= favoraretter

* http://d.hatena.ne.jp/snaka72/

== DESCRIPTION:

指定されたユーザーのふぁぼられを監視して、誰かからふぁぼられたら
Growlでお知らせします。

== FEATURES/PROBLEMS:

下記機能をバッチファイルとして提供してます
* サービスへの登録
* サービスからの登録解除
* サービスの開始
* サービスの停止

== REQUIREMENTS:

* nokogiri (>= 1.3.2)
* ruby_gntp (>= 0.1.1)
* win32-service (>= 0.6.1)

== INSTALL:

* gem install snaka-favoraretter --source http://gems.github.com/
